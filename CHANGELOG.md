# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-18

### Added
- Initial release of the `snap_share` plugin
- Support for sharing photos to Snapchat on iOS and Android
- Support for sharing videos to Snapchat (iOS only)
- Sticker overlay support for photos
- Caption and attachment URL support
- `isSnapchatInstalled()` utility method
- Custom `SnapShareException` for better error handling
- File existence validation before sharing
- Comprehensive documentation and examples
